About Supreme Linux

Supreme Linux is a Linux distribution based on Debian Sid, it is in experimental stage , it will be easy to use with graphics drivers preistalled and with utility center to manage your debian system easier , packages are gonna be bleeding edge always updating with latest versions of drivers and firmware. It uses the Gnome graphical environment and it will bring the best Gnome experience ot the users . We are gonna make some tweaks to accelerate 3d performance and battery efficiency on laptops and also have a software boutique so you can download applications/packages that are not available from Debian Sid Repositories. It will be a pretty stable Modern and fast Linux Distribution.
